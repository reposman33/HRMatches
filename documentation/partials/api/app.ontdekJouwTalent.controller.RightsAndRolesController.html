RightsAndRolesController
controller in module app.ontdekJouwTalent

Description

This controller contains functionality for usermanagement RightsAndRoles

Dependencies

  • $scope,roles,permissions

    Referring states: settings.usermanagement.rightsAndRoles

Methods

  • addRole()

    Called when user adds a role. The roles array is updated with a new role

  • deleteRole()

    Called when user deletes a role.

  • edit()

    Used in generic tableView.html to assign rights to a role

  • save()

    Used in generic tableView.html to save rights and roles
